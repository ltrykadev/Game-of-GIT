When you play the Game of GIT, you win or you rebase.

In the realm of version control, knowledge is power. Git is the sword every developer carries — yet few truly master it. Game of GIT is a leveled quest that takes you from empty directory to fearless rebasing, one command at a time.

You work with real Git commands inside a disposable sandbox. Nothing you type here can harm your real projects. Mistakes are not punishments — they are lessons. Every wrong turn reveals what Git is actually thinking.

Ten levels. Ten conceptual clusters. One terminal. The night is dark and full of merge conflicts — but you will not face them alone.
