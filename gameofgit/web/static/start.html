GAME OF GIT

How to Play

  • Type real git commands in the shell pane — exactly as you would in a terminal.
  • Each quest succeeds when the repository state matches the goal — there is no single "correct" command string.
  • Type ? and press Enter, or click Show next hint, to reveal a clue without penalty.
  • Only the commands relevant to the current level are available — the whitelist is shown in the quest pane.
  • The sandbox is disposable. Close the tab, lose the game, start fresh — no lasting damage, ever.

The night is dark and full of merges.
